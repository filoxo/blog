## Prereqs

1. [Install AWS cli](https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html) and get signed in
1. [Install Terraform](https://learn.hashicorp.com/tutorials/terraform/install-cli)

## Setup AWS

1. `terraform init`
1. `terraform plan`

## Create a Lambda function

1. `yarn add aws-lambda-fastify fastify`
